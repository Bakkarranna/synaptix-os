## Hook Patterns That Work

1. **Price Shock**: "A WEB STUDIO CHARGES $35,000... THE SAME BUILD NOW COSTS $12"
2. **Authority + Secret**: "My friend makes $1.2M/year. He sent me a video never meant to get out."
3. **Simple Hack**: "Sells $10,000 websites using ONE stupidly simple trick"
4. **Underdog Story**: "She was making $9/hr at McDonald's. Last week: $10,400."
5. **You're Doing It Wrong**: "I've been using Claude completely wrong for two years"
6. **Urgency/JOMO**: "Anthropic QUIETLY dropped their own official skills repo"
7. **Specific Number**: "6 Claude design skills", "12 AI websites", "30+ free resources"

## The Winning Post Formula

```
HOOK (1 tweet) — Controversial/underdog/shock statement with number
    ↓
CONTEXT (2-3 tweets) — Why this matters, who it's for, the problem
    ↓
SYSTEM BREAKDOWN (5-8 tweets) — Step-by-step, each tweet = 1 concept
    ↓
VISUAL PAYOFF (1 tweet) — Video/gif showing the result
    ↓
RESOURCES (2-3 tweets) — Links, templates, tools used
    ↓
CTA (1 tweet) — "Reply X for the playbook" or "Bookmark this"
```

## Dead Topics (Low Engagement)

- Generic "how I learned to code" stories
- Pure portfolio drops without process breakdown
- Simple tech stack announcements
- AI slop/fear-mongering content
- Overly technical code snippets without visual payoff

## Engagement Tactics

- "Reply 'X' and I'll send you the playbook" — ZEUS got 155 replies
- "Drop a 'Hi!' in the comments" — low effort, high volume
- "Which website would you add?" — open-ended question at thread end
- "What am I missing?" — implied expertise + community building
