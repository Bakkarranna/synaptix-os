## CROSS-CUTTING OBSERVATIONS

### Content Formula Patterns
1. **The "Build in Public" playbook** (Nick Buzz, Levels, Theo): Share process, failures, revenue transparently. Build trust through vulnerability + results.
2. **The "Deep Niche Authority" playbook** (Bruno Simon, Josh Comeau, Ahmad Shadeed): Obsess over one domain, produce best-in-class educational content, charge premium.
3. **The "Hot Take News" playbook** (Fireship, Theo): Fast reactions to industry news, strong opinions, personality-driven commentary.
4. **The "Before/After Roast" playbook** (Nick Buzz): Demonstrate value by critiquing others' work publicly. High virality, high credibility.

### Most Underserved Niches
1. **Three.js + AI integration**: Almost nobody covers using AI coding tools with Three.js/R3F. Massive gap.
2. **R3F for production**: Bruno's course covers pure Three.js. Almost no dedicated R3F production content (Wawa Sensei is closest but small).
3. **Web animation performance**: Josh Comeau covers CSS vs JS perf but nobody does deep dives on GSAP vs Framer Motion vs CSS for production.
4. **Agency owners who build in 3D**: Huy Nguyen is the closest with Webflow — no one covers Three.js/WebGL agency building.
5. **Cinematic portfolio building**: Divyant and Sujitkoji have the right idea but no content engine. Huge demand for "build a cinematic portfolio" tutorials.
6. **AI tools for creative devs**: Tidewave, Genspark Build, Enter Pro, Siteplay — nobody is making content about these tools for the creative dev audience.
7. **WebGPU for Three.js devs**: The platform shift is coming but no content creator is covering it yet.

### Highest Opportunity Content Gaps
- "Build a Three.js portfolio with AI coding agents" — zero content exists
- "GSAP vs Framer Motion vs CSS: When to use what" — no definitive guide
- "How creative dev agencies win clients" — no one covering this (Nick Buzz is productized services, not agency)
- "Three.js performance optimization for production" — partial coverage but no single source
- "AI website builders for creative portfolios: comprehensive review" — lots of tool reviews exist but not for creative dev audience
